# テンキーレジシステム 使用説明書

## 概要

テンキー（数字キーパッド）だけで操作できるシンプルなレジシステムです。  
商品コードと数量を入力してカートに追加し、お会計でお釣りを計算します。

---

## 起動方法

```bash
python3 register.py
```

Python 3 が動作する環境であれば、追加ライブラリのインストールは不要です。

---

## 基本的な操作の流れ

```
1. 商品コード(2桁) を入力 → [Enter]
2. 数量 を入力           → [Enter]
        ↑ 購入商品の数だけ繰り返す
3. [00] を入力           → [Enter]  ← お会計へ
4. お預かり金額 を入力   → [Enter]
5. お釣りが表示される
6. [Enter]               ← 次のお客様へ（1に戻る）
```

---

## 各フェーズの説明

### 商品登録フェーズ

画面上部にカートの内容が表示されます。

```
====================================================
         テンキーレジシステム
====================================================
   No  商品名              数量    単価     小計
----------------------------------------------------
    1  コーヒー               2     300      600
    2  おにぎり               1     150      150
----------------------------------------------------
  合計                                  ¥      750
====================================================
  商品コード(2桁)+[Enter] で商品追加
  [00]=会計へ  [98]=商品一覧  [99]=終了
====================================================
  商品コード >> _
```

**操作キー一覧**

| 入力 | 動作 |
|------|------|
| `01`〜`31` + Enter | 商品を選択し、数量入力画面へ進む |
| `00` + Enter | お会計フェーズへ進む（カートが空のときは不可） |
| `98` + Enter | 登録商品の一覧を表示する |
| `99` + Enter | システムを終了する |
| BS（バックスペース） | Enter を押す前であれば入力を訂正できる |

**数量入力**

商品コードを入力すると数量入力画面になります。

```
  選択商品: [01] コーヒー          ¥300 / 個
====================================================
  数量 >> _
```

- `1`〜`999` の整数を入力して Enter を押します。
- `0` や数字以外を入力するとエラーになり、もう一度入力できます。

---

### お会計フェーズ

`00` + Enter でこの画面に入ります。

```
  *** お会計 ***  合計: ¥750
====================================================
  お預かり金額 >> ¥_
```

- お客様からお受け取りした金額を入力して Enter を押します。
- 合計金額より少ない金額を入力するとエラーが表示されます。

**完了画面**

```
****************************************************
  合計金額  :  ¥       750
  お預かり  :  ¥     1,000
  お 釣 り  :  ¥       250
****************************************************
  ¥250 をお返しします。ありがとうございます！
====================================================
  [Enter] で次のお客様へ > _
```

Enter を押すと、カートがリセットされ次のお客様の対応が始まります。

## 商品を追加・変更する方法

`register.py` の先頭にある `PRODUCTS` 辞書を編集します。

```python
PRODUCTS = {
    "01": ("コーヒー", 300),
    "02": ("紅茶",     280),
    ...
}
```

**形式**

```python
"コード": ("商品名", 単価),
```

- **コード** : `"01"` 〜 `"97"` の2桁の文字列（`00`・`98`・`99` は予約済みで使用不可）
- **商品名** : 任意の文字列（18文字相当の表示幅を超えると列がずれることがあります）
- **単価** : 整数（円）

**追加例**

```python
"50": ("ホットドッグ", 380),
"51": ("フライドポテト", 320),
```

## エラーメッセージ一覧

| メッセージ | 原因と対処 |
|-----------|-----------|
| `!! 商品が登録されていません` | カートが空のまま `00` を押した。商品を追加してから会計へ進む |
| `!! [XX] 2桁の商品コードを入力してください` | 2桁以外の入力をした。正しいコードを入力し直す |
| `!! [XX] 登録されていない商品コードです` | 存在しないコードを入力した。`98` で商品一覧を確認する |
| `!! 数量は 1〜999 の整数で入力してください` | 0 以下・1000 以上・数字以外を入力した |
| `!! 正しい金額を入力してください` | お預かり金額に数字以外を入力した |
| `!! ¥XXX 不足しています` | お預かり金額が合計より少ない。正しい金額を入力し直す |

---

## 終了方法

- `99` + Enter を入力する
- または `Ctrl + C` を押す
