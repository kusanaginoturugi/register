# テンキーレジシステム 使用説明書

## 概要

テンキー（数字キーパッド）だけで操作できるシンプルなレジシステムです。  
商品コードと数量を入力してカートに追加し、お会計でお釣りを計算します。

---

## 起動方法

### Web版

`index.html` をブラウザで開きます。サーバーは不要です。

### Python版

```bash
python3 register.py
```

Python 3 が動作する環境であれば、追加ライブラリのインストールは不要です。

---

## Web版の画面

Web版は上から順に、入力表示、操作方法・カート・商品一覧のパネルが並びます。

商品一覧には商品登録フォームがあります。

| 入力欄 | 内容 |
|--------|------|
| コード | 2桁の商品コード |
| 商品名 | 表示する商品名 |
| 単価 | 1円以上の整数 |

登録した商品はブラウザの localStorage に保存され、画面をリロードしても残ります。`00`・`98`・`99` は予約済みコードのため登録できません。

---

## 基本的な操作の流れ

```
1. 商品コード(2桁) を入力 → [Enter]
2. 数量 を入力           → [Enter]
        ↑ 購入商品の数だけ繰り返す
3. [00] を入力           → [Enter]  ← お会計へ
4. お預かり金額 を入力   → [Enter]
5. お釣りが表示される
6. [Enter]               ← 次のお客様へ（1に戻る）
```

---

## 各フェーズの説明

### 商品登録フェーズ

Web版では、入力表示の下にカートの内容が表示されます。Python版では、画面上部にカートの内容が表示されます。

Python版の表示例:

```
====================================================
         テンキーレジシステム
====================================================
   No  商品名              数量    単価     小計
----------------------------------------------------
    1  コーヒー               2     300      600
    2  おにぎり               1     150      150
----------------------------------------------------
  合計                                  ¥      750
====================================================
  商品コード(2桁)+[Enter] で商品追加
  [00]=会計へ  [98]=商品一覧  [99]=終了
====================================================
  商品コード >> _
```

**操作キー一覧**

| 入力 | 動作 |
|------|------|
| `01`〜`97` + Enter | 商品を選択し、数量入力画面へ進む |
| `00` + Enter | お会計フェーズへ進む（カートが空のときは不可） |
| `-` + Enter | 直前の商品をカートから削除する |
| `+` + Enter | 直前の商品の数量を変更する |
| `*` | Web版: レシート履歴 → 商品別集計 → 閉じる |
| `/` | Web版: 操作方法・商品一覧の表示を切り替える |
| `98` + Enter | Python版: 登録商品の一覧を表示する |
| `99` + Enter | Python版: システムを終了する |
| BS（バックスペース） | Enter を押す前であれば入力を訂正できる |

**数量入力**

商品コードを入力すると数量入力画面になります。

```
  選択商品: [01] コーヒー          ¥300 / 個
====================================================
  数量 >> _
```

- `1`〜`999` の整数を入力して Enter を押します。
- `0` や数字以外を入力するとエラーになり、もう一度入力できます。

---

### お会計フェーズ

`00` + Enter でこの画面に入ります。

```
  *** お会計 ***  合計: ¥750
====================================================
  お預かり金額 >> ¥_
```

- お客様からお受け取りした金額を入力して Enter を押します。
- 合計金額より少ない金額を入力するとエラーが表示されます。

**完了画面**

```
****************************************************
  合計金額  :  ¥       750
  お預かり  :  ¥     1,000
  お 釣 り  :  ¥       250
****************************************************
  ¥250 をお返しします。ありがとうございます！
====================================================
  [Enter] で次のお客様へ > _
```

Enter を押すと、カートがリセットされ次のお客様の対応が始まります。

## 商品を追加・変更する方法

### Web版

商品一覧の登録フォームから、商品を追加できます。

1. コード、商品名、単価を入力する
2. `登録` を押す
3. 商品一覧に追加される

カートに登録した商品は、カート行の `×` ボタンを押すと確認後に削除できます。テンキー操作では `-` + Enter で直前の商品を削除できます。

追加した商品はブラウザの localStorage に保存されます。

ヘッダーの **商品管理** から、商品マスターCSVのダウンロード、CSVインポート、デフォルト商品データへの復元もできます。

CSVフォーマット:

```csv
コード,商品名,単価
01,コーヒー,300
02,紅茶,280
```

### Python版

`register.py` の先頭にある `PRODUCTS` 辞書を編集します。

```python
PRODUCTS = {
    "01": ("コーヒー", 300),
    "02": ("紅茶",     280),
    ...
}
```

**形式**

```python
"コード": ("商品名", 単価),
```

- **コード** : `"01"` 〜 `"97"` の2桁の文字列（`00`・`98`・`99` は予約済みで使用不可）
- **商品名** : 任意の文字列（18文字相当の表示幅を超えると列がずれることがあります）
- **単価** : 整数（円）

**追加例**

```python
"50": ("ホットドッグ", 380),
"51": ("フライドポテト", 320),
```

## レシート履歴と商品別集計（Web版）

`*` キーで、レシート履歴 → 商品別集計 → 閉じる、の順に切り替えます。商品別集計では、商品名・合計数・合計金額を表示します。

| 表示 | 内容 |
|------|------|
| レシート履歴 | レシート番号、日時、合計、明細 |
| 商品別集計 | 商品名、合計数、合計金額 |

どちらの画面にも **Excel用に保存** ボタンがあります。

- レシート履歴CSV: レシート件数、合計売上、明細を保存
- 商品別集計CSV: 商品名、合計数、合計金額を保存

レシート履歴は localStorage に保存されるため、画面をリロードしても残ります。会計途中のカートや入力途中の値は一時状態のため、リロードすると消えます。

## エラーメッセージ一覧

| メッセージ | 原因と対処 |
|-----------|-----------|
| `!! 商品が登録されていません` | カートが空のまま `00` を押した。商品を追加してから会計へ進む |
| `!! [XX] 2桁の商品コードを入力してください` | 2桁以外の入力をした。正しいコードを入力し直す |
| `!! [XX] 登録されていない商品コードです` | 存在しないコードを入力した。Web版では商品一覧、Python版では `98` で商品一覧を確認する |
| `!! 数量は 1〜999 の整数で入力してください` | 0 以下・1000 以上・数字以外を入力した |
| `!! 正しい金額を入力してください` | お預かり金額に数字以外を入力した |
| `!! ¥XXX 不足しています` | お預かり金額が合計より少ない。正しい金額を入力し直す |

## 終了方法

Web版はブラウザのタブを閉じます。

Python版は以下のいずれかで終了します。

- `99` + Enter を入力する
- `Ctrl + C` を押す
